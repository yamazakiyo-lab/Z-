## 1. はじめに

### 1-1. TSEG WORKSとは

会社の写真・動画・過去の指令書PDF、工番、納入先、部品在庫を、ブラウザからまとめて検索できる社内アプリです。目的の資料や在庫を、工番・工事名・納入先・部品の型式などから素早く探せます。

ホーム画面から用途別のメニューを選んで使います。メニューは今後も追加されます。

現在のメニューは次の4つです。①FMP SEARCH（写真・動画・指令書PDF検索）／②工番検索／③納入先検索／④部品在庫検索（寄居・綾瀬の貯蔵品）。

### 1-2. アクセス方法

- 会社で配布されているTSEG WORKSのURLをブラウザで開きます(タブレット・スマートフォン・PCどれでも可)。

- 初回アクセス時は、会社の Microsoft アカウント(Entra ID)でサインインします(Googleアカウントは使いません)。

- スマホ・タブレットではブックマークやホーム画面追加をしておくと便利です。

- 社外から使う場合は、会社のネットワーク接続(VPN等)が必要なことがあります。

## 2. ホーム画面とメニュー

最初に開くと「ホーム(玄関ページ)」が表示されます。使いたい検索メニューをクリックして進みます。左側のサイドバー(スマホでは左上の「≫」)からも各メニューに移動できます。

| メニュー | 何を探すか |
| --- | --- |
| 🔍 FMP SEARCH | 写真・動画・過去の指令書PDF |
| 🔎 工番検索 | 工事名・納入先・工番の一部から「工番」を探す(完成/未成でも絞れる) |
| 🏢 納入先検索 | 納入先(会社名)から、その取引先の工番・写真 |
| 📦 部品在庫検索 | 貯蔵品(寄居・綾瀬)の在庫を型式・品名・メーカー等から探す |

## 3. FMP SEARCH（写真・動画・指令書PDF検索）

### 3-1. 検索のしかた

1. 「キーワード」に、工番・作業・コメントなどを入力します(スペース区切りで複数語のAND検索)。

2. 必要に応じて「納入先名」「請求先名」でも絞り込めます(部分一致)。

3. 「🔍 検索」を押すと結果が一覧表示されます。

### 3-2. 絞り込みフィルタ

- 種別: 📷写真 / 🎬動画 / 📄指令書PDF(未選択ならすべて表示)

- フェーズ: 🟦B1 着手前 / 🟩B2 着手中 / 🟨B3 出荷以降 / 🟥B4 整理前

- 表示件数: 一覧に出す最大件数を調整できます。

### 3-3. 写真・動画・PDFの見かた

- 写真: サムネイルが表示され、「🔍 拡大」でフルサイズ表示できます。

- 動画: 「🎬 動画を開く / ダウンロード」から別タブ表示またはダウンロードで再生できます(ブラウザで直接再生できない形式でも、ダウンロードすればPCの動画プレーヤーで見られます)。

- 指令書PDF: 「📄 PDFを開く」で別タブに表示されます。

各結果には、工番・工事名・納入先・請求先・撮影日・メモ、元ファイルの保存パスも表示されます。

## 4. 工番検索

「工番が分からないが、工事名や納入先は分かる」ときに、工番を探すためのメニューです。

### 4-1. 検索のしかた

1. 「キーワード」に、工事名・納入先・工番の一部を入力します(例: トータス / 高千穂 / 4031)。

2. 該当する工番が一覧表示されます(工番・工事名・納入先・写真枚数)。

※ 機械名は工事名に含まれることが多いため、機械名でも工事名検索でヒットします。

※ 一覧に出るのは「写真がある工番」だけです。そのため、見つけた工番で写真を見れば必ず表示されます。

### 4-2. 完成 / 未成で絞り込む

- 「状態で絞り込み: すべて / 完成 / 未成」で、案件の状態で絞れます。

- 各工番には 🟢完成 / 🔴未成 のバッジが付きます(工事完成日の有無で判定)。

### 4-3. 写真へジャンプ

- 一覧の「📷 写真を見る」を押すと、FMP SEARCH に切り替わり、その工番の写真が自動表示されます。

## 5. 納入先検索

納入先(注文者)の会社名から、その取引先の工番・写真をまとめて探すメニューです。

1. 「納入先名」に会社名の一部を入力します(例: 高千穂 / 須藤)。

2. 該当する納入先が一覧表示されます(工番数・写真件数つき)。

3. 納入先を開く(展開する)と、その取引先の工番・工事名・写真枚数が並びます。

4. 「📷 写真を見る」で、その工番の写真へジャンプできます。

## 6. 部品在庫検索

貯蔵品(寄居・綾瀬)の在庫を、型式・品名・メーカー・仕入先・棚番から探すメニューです。

### 6-1. 検索のしかた

1. 「キーワード」に、型式・品名・メーカー・仕入先・棚番のいずれかの一部を入力します(例: ベアリング / SMC / 6203 / TN1)。

2. 必要なら「カテゴリで絞り込み」で、在庫カテゴリを選べます。

3. 該当する部品が表で表示されます。

### 6-2. 表示される項目

- カテゴリ / 棚番 / 型式・品名 / 用途・仕様 / メーカー / 仕入先 / 数量 / 見積単価

※ 画面上部に「在庫データ更新日」と「登録品目数」が表示されます。在庫表の更新後に反映されます。

## 7. 使い方のヒント・注意

- スマートフォンでは、表(部品在庫など)が画面幅で切れることがあります。表を横にスクロールすると数量・見積単価まで見られます。

- うまく出ないときは、キーワードを短く・部分的にすると見つかりやすくなります。

- 工番検索・納入先検索は「写真がある工番」を対象に集計しています。写真がまだ無い新しい工番は出ないことがあります。

- 在庫・工番の状態(完成/未成)などのデータは、元データの更新後に反映されます。すぐ反映されない場合は時間をおいて再読み込みしてください。
